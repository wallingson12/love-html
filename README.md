# 💖 Projec-crush

Um mini site criado para expressar sentimentos de forma criativa, utilizando **HTML**, **CSS** e **JavaScript**.

---

## ✨ Funcionalidades

- Mensagem personalizada para alguém especial.  
- Quiz interativo com validação de resposta.  
- Imagens e animações para deixar a experiência mais divertida.

---

## 🚀 Como Usar

1. Clone o repositório:

git clone https://github.com/wallingson12/love-html.git
cd love-html-main
---
2. Abra o site no navegador:
Clique em templates/index.html
